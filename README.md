# dashboard-frontend

## Install & Setup
```
git clone https://github.com/positron48/dashboard-frontend.git 
cd dashboard-frontend
yarn install
cp .env .env.local
```
edit VUE_APP_API_HOST

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
